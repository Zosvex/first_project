Термины

  1. Антивирус — программа, предназначенная для поиска и удаления вредоносного программного обеспечения.
  2. Вирус — вредоносная программа, способная самораспространяться и повреждать данные.
  3. Обновление — установка новой версии программы с исправлениями или улучшениями.
  4. База вирусов — набор сигнатур, по которым антивирус определяет вредоносное ПО.
  5. Лицензия — разрешение на использование программного продукта в течение определённого срока.

Термины

  • Драйвер — программный компонент, обеспечивающий взаимодействие ОС с оборудованием.
  • Фаервол — система фильтрации сетевого трафика для защиты компьютера от атак.
  • Резервное копирование — создание копии данных для их последующего восстановления.
  • Облачное хранилище — удалённый сервер для хранения и синхронизации данных.

Термины

Шифрование
Процесс преобразования данных в недоступную для чтения форму для защиты информации.
Расшифровка
Обратный процесс, возвращающий зашифрованные данные в исходный вид.
Ключ шифрования
Набор параметров, используемых при шифровании и расшифровке данных.
Термины
  • Интерфейс пользователя — внешний вид и логика взаимодействия программы с человеком:
    1. Графический (GUI)
    2. Командный (CLI)
    3. Веб-интерфейс
  • Отладка — процесс поиска и исправления ошибок в программном коде.
  • Компилятор — программа, переводящая код на языке программирования в машинный код.
Термины
API
Интерфейс для взаимодействия программ между собой.
Баг
Ошибка в программе, вызывающая неправильное поведение.
Релиз
Официальный выпуск готовой версии программного продукта.
Репозиторий
Хранилище исходного кода и версий программ.
